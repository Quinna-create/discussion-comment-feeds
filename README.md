# discussion-comment-feeds

## Quick Start

### GitHub Pages Usage

This repository includes a static widget that can be published via GitHub Pages:

**Live Widget URL:** https://Quinna-create.github.io/discussion-comment-feeds/discussion-widget.html

#### Enabling GitHub Pages

To enable GitHub Pages for this repository:

1. Go to repository **Settings**
2. Navigate to **Pages** section
3. Under **Source**, select:
   - Branch: **main**
   - Folder: **/docs**
4. Click **Save**

After enabling, the widget will be available at the URL above.

**Note:** Ensure all video and API URLs used in the widget are HTTPS and the API allows CORS from the Pages origin.
